Exercise: Learn FORTH

Open the following: In-browser FORTH environment
FORTH is a stack-based programming language invented in the 60s, but that is still used to this day in micro-controllers and elsewhere.
It is a great way to learn about stacks. You can define a new function like this:
: times-three 3 * ;
This function expects a single argument on the stack, pushes a 3, and then calls * to multiply the two numbers. You can call it like this: 
4 time-three
Then use . to print out the result (12), or just "drop" to throw it away.
Your exercise is to create a function that evaluates the following:
$$f(x,y) = (x + 5)^2 - y$$
You should be able to test it like this: 
3 5 f .
And it should print: 
59
